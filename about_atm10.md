# All the Mods 10について

All the Mods 10は、ATMTeamにより開発されている、多数のMODを詰め込んだMODパックである。[こちら](https://www.curseforge.com/minecraft/modpacks/all-the-mods-10)からダウンロードが可能である。入っているMODの総数は、基礎MODなどを含めると476(2025年12月2日現在最新のバージョン5.2)で、大規模MODパックに属する。基礎となるMinecraftのバージョンは1.21.1、MODローダーはNeoForgeである。

この手の大規模MODパックとしては、バランスが優れており、かつ丁寧なクエストラインが整備されていることもあり、Minecraft 1.21.1のMOD環境を初めて触る人にとっても、パソコンのスペックが許すならば最適であるといえる。

ただ、大規模MODパックの定めとして、メモリなどのスペックは相応に要求されるという問題はある。目安として、必要割り当てメモリ量として8GB(8192MB)が定められており、16GB(16384MB)が推奨されている。このため、低スペックのパソコンでは、動かすのも難しいだろう。

このMODパックは、GitHub上で開発がなされており、[こちら](https://github.com/AllTheMods/ATM-10)でソースも公開されているため、興味がある人は見てみるとよいだろう。

## All the Mods 10の目標

All the Mods 10には、明確にこれを目指さなければならないという、必須の目標は存在しない。しかしながら、一般に「ATMスター」と呼ばれるこのMODパック固有のアイテムの作成が、本書で説明するクエストラインのうち、メインのものの目標となっている。
